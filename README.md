🎮 Simon Says Game

An interactive Simon Says memory game built using HTML, CSS, and JavaScript. This game challenges users to remember and repeat an increasing sequence of colors, improving memory and focus in a fun and engaging way.

📌 Features
🎯 Interactive gameplay
🎨 Smooth animations and UI effects
📈 Increasing difficulty with each level
🔁 Restart functionality
📊 Real-time score display
💻 Responsive design
🧠 How to Play
Start the game by pressing any key or clicking on the screen
Watch the color sequence carefully
Repeat the sequence by clicking the buttons
Each level adds a new color to the sequence
A wrong click ends the game
Try to achieve the highest score possible!
